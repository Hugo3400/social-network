### For Production Deployment

1. **Clone on your VPS**
   ```bash
   ssh user@your-vps
   git clone https://github.com/Hugo3400/social-network.git
   cd social-network
   ```

2. **Run the installer**
   ```bash
   ./install.sh
   ```

3. **Configure domain** (optional)
   - Update `nginx.production.conf` with your domain
   - Set up SSL with Let's Encrypt
   - Point domain to your VPS

4. **Open browser and complete wizard**
   - Navigate to `http://your-domain.com/setup`
   - Follow the 5-step wizard
   - Done!
